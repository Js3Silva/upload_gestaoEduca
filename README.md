# Gestão Educa

Desenvolvimento de um sistema de gestão de projetos customizado para o Instituto Educa Cidades, com foco na centralização, organização e controle de dados operacionais e gerenciais.

A proposta visa otimizar processos internos, aumentar a produtividade e viabilizar a gestão eficiente de dados, solucionando a atual ausência de uma ferramenta tecnológica adequada no instituto.
